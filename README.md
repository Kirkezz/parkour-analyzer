## Preparation

Install npm, rust and tauri using your system's package manager.
On Windows, go to https://nodejs.org/ and https://win.rustup.rs/ and follow the instructions.

## Build
On Linux/Windows:
```
cd parkour-tauri
npm install
npm run tauri build
cd src-tauri/target/release/bundle/
```
